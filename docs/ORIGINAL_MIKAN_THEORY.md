# ミカン定理：原典

> 本文書は、noteで連載中の小説「果物世界」における主人公の手記から抜粋したものです。
> Mikan 3-Currency clr Index（MCI）の理論的原典であり、果物を通貨に見立てた寓話的な形式で為替理論を解説しています。
>
> **果物記号**：🍈（メロン）= 基軸通貨A、🌰（栗）= 主要通貨B、🍊（オレンジ）= 周辺通貨C

---

## ミカン定理：体系的整理

### 1. 機能定義：ミカン定理とは何か

**一言で**：3通貨のPPP乖離を、相対的な位置関係として整理・可視化する座標変換ツール。

**入力**：
- d(PPP,AB)：A通貨とB通貨のPPP乖離率
- d(PPP,AC)：A通貨とC通貨のPPP乖離率

**出力**：
- m[A], m[B], m[C]：3通貨かご内での相対ポジション
- E[A], E[B], E[C]：PPP基準での絶対座標（任意）

**本質**：既存のPPP情報を「3通貨の位置関係」として再配置する。

### 2. できること

#### 2-1. 三通貨の相対バランスを一目で可視化

**従来のPPP分析**：
- A-B のチャート
- A-C のチャート
- → 別々に見る必要

**ミカン定理**：
- m[A], m[B], m[C] の3本線
- → 一枚で相対関係が分かる
  - 誰が一番浮いているか
  - 誰が底に沈んでいるか
  - 真ん中ポジションは誰か

例：「🍊が最も深く沈んでおり、🌰が🍈と🍊の中間に位置している」という構図を直感的に把握できる。

#### 2-2. 時系列化でストレスの移動を捉える

3本線チャートの時系列で見えるもの：

**クロス（交差）**
- m[B]がm[C]を下抜け
- → 主要通貨が周辺通貨水準へ転落

**距離の開閉**
- m[A]と他2通貨の距離が拡大
- → ストレス集中／基軸通貨の孤立

**ばらつき σ(m[*])**
- σ拡大 → 緊張・歪みの増大
- σ縮小 → ストレス拡散・均衡化

どの通貨にストレスが溜まり、どこへ逃げているかを図解できる。

#### 2-3. 直感的な理解の補助

PPP分析でよくある疑問：
- 「PPP割安なのになぜ円安が続くのか」
- 「どの通貨が本当に割安なのか」

**ミカン定理での説明**：
- 絶対軸（E[*]）：世界PPP基準での位置
- 相対軸（m[*]）：3通貨間での位置

この2つの視点を分けることで、一見矛盾する現象を整理できる。

### 3. できないこと

#### 3-1. 絶対的な割高・割安の断定

ミカン定理が示すのは：
- 3通貨の中での相対位置
- PPP乖離の再配置

示さないもの：
- 理論的に正しい均衡レート
- ❌ 「m[B] = -0.6だから本質的に超割安で必ず戻る」

#### 3-2. 4通貨以上の同時分析

制約：
- 基本的に3通貨前提の枠組み
- 4通貨以上は別の方法が必要

注意：
- どの3通貨を選ぶかでm[*]の値が変わる
- 「かごXの中でのB」と「かごYの中でのB」は別物
- ❌ 「世界全体の通貨評価がミカン定理で分かる」

#### 3-3. 単独での売買シグナル

ミカン定理が示すのは：
- PPPベースの静的歪みの整理
- フロー・金利・政治イベントは含まない

- ❌ 「m[C] = -1.5だからロング」
- ✅ 「歪みがどこに溜まっているか」を把握する補助

### 4. 技術的注意事項

#### 4-1. バスケット依存性

- m[*]は「この3通貨かごの中で」の相対位置
- かごが変われば平均も変わる
- この3通貨セットの中での相対評価として理解する必要がある。

#### 4-2. PPPの限界を継承

PPP自体の問題：
- Balassa-Samuelson効果
- 非貿易財の存在
- 構造的格差・リスクプレミアム

ミカン定理はPPPを元データとするため、これらの限界をそのまま継承する……。

#### 4-3. 絶対軸の絶対性は限定的

- E[*]は「差」から復元できる範囲の座標
- 3通貨が揃って世界に対してズレる共通成分は識別不可能
- 「絶対軸」は「この3通貨＋PPPの前提内での絶対座標」という意味。しかし、ミカン定理はその弱点を克服しうる発展性を備える

### 5. 位置づけ

ミカン定理は、3通貨のPPP乖離を相対的な位置関係として整理・可視化するツールです。

**得意なこと**：この3通貨の中で誰が浮きすぎていて、誰が沈みすぎているのかを時系列で直感的に把握すること。

**扱わないこと**：世界全体に対する絶対的な適正水準、多数通貨の同時評価。

**実務での位置づけ**：通常のPPP・REER分析、金利・フロー・政治要因の分析などと組み合わせて、「3通貨間でどこに歪みが集中しているか」を補助的に把握するためのレンズ。３本線グラフ化してRSI的な視覚化が効果的です

## 理論解説

# ミカン定理（PPP版）

## 三つの通貨果物 🍈🌰🍊 と Pan-fruit Price Parity（PPP）

---

### 第1章　ミカン定理とは何か

果物世界では、果物そのものが通貨として用いられています。様々な通貨果物が流通していますが、今回は果物世界を代表する基軸通貨🍈、主要通貨の一つである🌰、そして周辺通貨🍊という三つの通貨果物に焦点を当て、その割高・割安を整理するための理論を解説します。これが「ミカン定理」です。

この理論は、果物世界の二つの主要機関と、それらが算出する指標に基づいています。

**OECD（Order of Eternal Citrus Dynamics：永続柑橘動態機構）**
果物世界の長期的な価格均衡を研究する中心的な機関です。

**ICP（Interfruit Comparison Program：国際果実比較プログラム）**
各国統計局の物価調査を取りまとめ、PPP算出に必要なデータを提供する機関です。

**PPP（Pan-fruit Price Parity：汎果物価均衡）**
OECDとICPが、各国統計局の物価調査を基に算出する「汎果物価均衡レート」を指す指標です。

PPPとは、「同じ生活コストかごを支払うのに必要な通貨果物の量が、通貨ごとにどれくらい違うか」を基準化した長期フェアバリュー指標です。ICPが集めた統計データをもとに、OECDとともにこの均衡レートを算出しています。

また、著名な投資分析機関であるGS（Golden Seed：ゴールデン・シード社）は、歴史的なデータ分析から重要な発見をしています。

- PPPから大きく乖離した通貨果物は、時間がたつとどこかのタイミングでPPP方向に収束してきた
- その収束の道筋は、金利差、資本フロー、政治ショック、投機などに左右される

GSはこの経験則を市場レポートの根幹に据え、PPP乖離を重要な分析指標として用いています。

ミカン定理は、このPPPを「絶対軸」として採用し、三つの通貨果物 🍈（A）・🌰（B）・🍊（C）の力関係を、数式で一貫して説明するための枠組みです。

---

### 第2章　三つの通貨果物とレートの基本設定

今回分析する三つの通貨果物は以下の通りです。

- **A：🍈（メロン）** - 果物世界の基軸通貨
- **B：🌰（栗）** - 果物世界の主要通貨の一つ
- **C：🍊（オレンジ）** - 周辺通貨

ミカン定理では、🍈を基軸として、以下のように表記します。

- S_AB = 1🍈あたりの🌰の実勢レート（スポット）
- S_AC = 1🍈あたりの🍊の実勢レート（スポット）

同様に、OECDとICPが算出するPPP（Pan-fruit Price Parity）に基づく「長期フェアバリュー（汎果物価均衡レート）」を次のように表記します。

- F_PPP_AB = 1🍈あたりの🌰のPPPフェアバリュー
- F_PPP_AC = 1🍈あたりの🍊のPPPフェアバリュー

S_AB、S_ACは市場で日々変動しますが、F_PPP_AB、F_PPP_ACは物価水準に基づいているため、基本的にはゆっくりとしか動きません。

---

### 第3章　PPP乖離率 d(PPP,AB), d(PPP,AC)

PPPからどれだけズレているかを測るために、「PPP乖離率」を次のように定義します。

```
d(PPP,AB) = ln( S_AB / F_PPP_AB )
d(PPP,AC) = ln( S_AC / F_PPP_AC )
```

ここでlnは自然対数です。

**意味の解釈**：

- **d(PPP,AB) > 0 の場合**
  - 実勢レートS_ABがPPPより大きい
  - → 🍈が🌰に対して割高側、または🌰が🍈に対して割安側

- **d(PPP,AB) < 0 の場合**
  - 実勢レートS_ABがPPPより小さい
  - → 🍈が🌰に対して割安側、または🌰が🍈に対して割高側

同様にd(PPP,AC)は、🍈と🍊の関係に対応します。

OECD、ICP、そしてGSは、このd(PPP,AB)、d(PPP,AC)を「果物世界における長期均衡からの乖離率」として扱っています。GSの歴史分析では、「この乖離率は長期的に平均回帰する（ゼロ付近に戻る）傾向がある」という事実が確認されています。

---

### 第4章　PPP絶対軸としての通貨歪み E[A], E[B], E[C]

PPPを絶対基準として、「果物世界全体から見た通貨果物の高さ・低さ」をE[*]で表します。

- E[A] = 🍈の絶対歪み
- E[B] = 🌰の絶対歪み
- E[C] = 🍊の絶対歪み

二通貨ペアのPPP乖離率は、これらの差として書けます。

```
d(PPP,AB) = E[A] - E[B]
d(PPP,AC) = E[A] - E[C]
```

この2つの式から、E[A]、E[B]、E[C]は「定数分のずれ（全体シフト）」を除けば一意に定まります。

ミカン定理では、🍈を「国際基準的な通貨果物」とみなして、慣例的に

```
E[A] = 0
```

を選びます。このとき、

```
E[B] = - d(PPP,AB)
E[C] = - d(PPP,AC)
```

となります。

E[B]がマイナスで大きいほど、「果物世界全体の視点で🌰が割安で放置されている」ことを意味し、E[C]も同様に、「🍊が世界目線でどれほど割安か」を表します。

---

### 第5章　三通貨かご内の相対偏位 m[A], m[B], m[C]

E[*]は「果物世界全体から見た絶対的な高さ」ですが、今回分析する🍈🌰🍊という三つの通貨果物に限定した視点では、

「この三つの通貨果物の中で、誰が浮いていて、誰が沈んでいるのか」

という相対的な位置関係の方が重要になることが多くあります。

そこで、三通貨かごの平均歪みを

```
E_bar = ( E[A] + E[B] + E[C] ) / 3
```

と定義し、各通貨果物のかご内相対偏位（位置）を

```
m[A] = E[A] - E_bar
m[B] = E[B] - E_bar
m[C] = E[C] - E_bar
```

と定義します。

このとき、必ず

```
m[A] + m[B] + m[C] = 0
```

が成り立ちます。これは、あくまで「三つの通貨果物のかご内で平均をゼロにそろえた座標を使っている」というだけの数学的性質です。果物世界全体で割高と割安が必ず相殺されるといった経済的な保存則を意味するものではありません。

m[A]がプラスで大きいほど、「この三つの通貨果物の中で🍈が浮いている（割高）」ことを意味し、m[C]がマイナスで大きいほど、「🍊が沈んでいる（割安）」ことを意味します。m[B]がゼロ近辺であれば、「🌰はこの三通貨の中で中立ポジション」にいると解釈できます。

---

### 第6章　三角分解公式：m[A], m[B], m[C] を d(PPP,AB), d(PPP,AC) から求める

第4章と第5章の定義から、以下の関係式が成り立ちます。

```
d(PPP,AB) = E[A] - E[B]
d(PPP,AC) = E[A] - E[C]

m[A] = E[A] - E_bar
m[B] = E[B] - E_bar
m[C] = E[C] - E_bar

かつ E_bar = ( E[A] + E[B] + E[C] ) / 3
```

これらを連立すると、m[*]はd(PPP,AB)、d(PPP,AC)だけで表せます。

**三角分解公式**：

```
m[A] = ( d(PPP,AB) + d(PPP,AC) ) / 3
m[B] = ( -2 * d(PPP,AB) + d(PPP,AC) ) / 3
m[C] = ( d(PPP,AB) - 2 * d(PPP,AC) ) / 3
```

この三角分解公式により、

- OECDとICPが公表するPPP乖離率（d(PPP,AB)、d(PPP,AC)）さえあれば
- 三つの通貨果物🍈🌰🍊がこの三通貨かごの中でどの位置にいるか（m[A]、m[B]、m[C]）が自動的に決まる

ことになります。

GSは、実務上はd(PPP,AB)、d(PPP,AC)の時系列をチャートにし、その裏側にこのm[*]の動きを意識しながら、「どの通貨果物が行き過ぎているか」「どこに是正圧力がたまりつつあるか」を評価しています。

### 第7章　2025-11-17 時点の数値例（PPP 適用）

ここでは果物歴2025年11月17日時点のデータを用いて、ミカン定理の計算を一度フルに展開します。

**実勢レート（スポット）**

```
S_AB = 154.536 （1🍈 あたりの 🌰）
S_AC = 42.3248 （1🍈 あたりの 🍊）
```

**PPP フェアバリュー（Pan-fruit Price Parity）**

```
F_PPP_AB = 93.52 （1🍈 あたりの 🌰 の PPP）
F_PPP_AC = 16.51 （1🍈 あたりの 🍊 の PPP）
```

#### 7.1　PPP乖離率 d(PPP,AB), d(PPP,AC)

```
d(PPP,AB) = ln( S_AB / F_PPP_AB )
          = ln( 154.536 / 93.52 )
          ≈ ln( 1.65 )
          ≈ 0.5023

d(PPP,AC) = ln( S_AC / F_PPP_AC )
          = ln( 42.3248 / 16.51 )
          ≈ ln( 2.56 )
          ≈ 0.9414
```

**解釈**：

- **d(PPP,AB) ≈ 0.5023**
  - 指数に戻すとexp(0.5023) - 1 ≈ +0.652
  - つまり、🍈側から見ると🌰に対してPPPより約+65%割高です。逆に🌰側から見ると🍈に対して約40%割安となります（65%割高の逆数をとると約40%割安になります）。

- **d(PPP,AC) ≈ 0.9414**
  - exp(0.9414) - 1 ≈ +1.563
  - つまり、🍈側から見ると🍊に対してPPPより約+156%割高です。🍊側から見ると🍈に対して60%超の超割安状態です。

#### 7.2　PPP絶対軸での通貨歪み E[A], E[B], E[C]

E[A]は基準として0とします。

```
E[A] = 0
E[B] = - d(PPP,AB) = -0.5023
E[C] = - d(PPP,AC) = -0.9414
```

**絶対軸における解釈**：

- **E[A] = 0** → 🍈は「世界標準」の高さ付近にある（基準として扱う）
- **E[B] ≈ -0.5023** → 果物世界のPPP基準から見ると🌰は約40%割安
- **E[C] ≈ -0.9414** → 果物世界のPPP基準から見ると🍊は約60%割安の超ディスカウント通貨

#### 7.3　三通貨かご内の相対座標 m[A], m[B], m[C]

**三通貨かごの平均歪み**：

```
E_bar = ( E[A] + E[B] + E[C] ) / 3
      = ( 0 - 0.5023 - 0.9414 ) / 3
      = -0.4812 （概算）
```

**相対座標**：

```
m[A] = E[A] - E_bar
     = 0 - ( -0.4812 )
     = +0.4812

m[B] = E[B] - E_bar
     = -0.5023 - ( -0.4812 )
     = -0.0210 （ほぼゼロ）

m[C] = E[C] - E_bar
     = -0.9414 - ( -0.4812 )
     = -0.4602
```

**チェック**として：

```
m[A] + m[B] + m[C]
= 0.4812 - 0.0210 - 0.4602
≈ 0
```

となり、三通貨かごのゼロ和条件が満たされています。

### 第8章　数値例のまとめと直感的な読み方

**絶対軸（PPP）で見た場合**：

- 🍈（A）は基準（E[A] = 0）
- 🌰（B）は世界目線で約40%割安（E[B] ≈ -0.50）
- 🍊（C）は世界目線で約60%割安（E[C] ≈ -0.94）

つまり、PPPの観点からは「🍈だけが割高／標準側にいて、主要通貨である🌰も周辺通貨である🍊もどちらも物価に比べて安く評価されている」という構図になります。特に🍊は深く沈んでいます。

**一方、三通貨かご内の相対座標（m[*]）で見ると**：

- **m[A] ≈ +0.48** → この三つの通貨果物の中で🍈が大きく浮いている（割高）
- **m[B] ≈ -0.02** → 🌰はこの三通貨の中では「ほぼ平均ポジション」
- **m[C] ≈ -0.46** → 🍊がかごの底で大きく沈んでいる（割安）

ここで重要なのは、視点の違いによって見え方が変わるという点です。

- 絶対軸E[*]では「主要通貨🌰も周辺通貨🍊も世界目線でかなり割安」
- 相対軸m[*]では「🍈が浮き、🍊が沈み、🌰は真ん中に見える」

この見え方の違いは、極端な🍊安が「かご全体の平均E_barを引き下げてしまう」ため、相対的に🌰が平均に近づいて見えるという、純粋な座標の効果です。これはモデルの限界ではなく、座標系の性質によるものです。

### 第9章　果物世界におけるPPPの歴史的性質

果物世界における通貨果物の動きを整理すると、次のようになります。

**PPP（Pan-fruit Price Parity）とは**

OECDとICP（Interfruit Comparison Program）が、各国統計局の物価調査を基に算出する果物世界の「汎果物価均衡ライン」です。GSが市場分析で用いる重要な指標となっています。

各通貨果物は、短期的には投機や政策、戦争、災害などによってPPPから大きく乖離することがあります。しかし、GSの長期データベースによれば、

**「どれほど乖離しても、長期的にはPPP方向への平均回帰が一貫して観測されてきた」**

という経験則が歴史上繰り返し確認されています。

**三つの通貨果物🍈🌰🍊の特徴**

**🍈（A）**
基軸通貨果物。果物世界の相場の中心にいて、他の通貨果物がどう動いても、最終的にはPPPを基準に「割高・割安」が判断される対象です。

**🌰（B）**
果物世界の主要通貨の一つです。世界目線ではかなり割安な状態が続いています。今回分析した三通貨の中では「上に🍈、下に🍊がいる」ため、見かけ上は中立ポジションに見えやすい特性があります。🌰を使う地域の住民が果物世界全体の舞台（特に🍈圏）に出ると「物価が妙に高く感じる」という経験をすることになります。逆に🍈圏の人々が🌰圏を訪れると、「だいぶ安い」と感じることになるのです。

**🍊（C）**
周辺通貨。世界目線でも、この三通貨の中でも、もっとも深く沈んだ通貨果物です。物価に対して通貨の価値が低すぎるため、PPPから見て常に強い是正圧力を受けています。「改革」「通貨危機」「奇跡の復活」など、ドラマの主役になりやすいポジションです。

**ミカン定理の使い道**

- 果物世界全体の「絶対軸」で通貨果物を見るときは**E[A]、E[B]、E[C]**を使う
- 今回分析した三通貨の「相対軸」で見るときは**m[A]、m[B]、m[C]**を使う
- マーケットの分析では**d(PPP,AB)、d(PPP,AC)**のチャートを基礎とし、その裏にあるE[]やm[]の意味をGSがレポートでかみ砕いて説明する

こうした構造を理解することで、

- 「なぜ🍊がここまで安く放置されているのか？」
- 「なぜ🍈圏の人々が🌰圏に行くと"物価が安く感じる"のか？」
- 「なぜ主要通貨である🌰を使う地域の人々が🍈圏に出ると"物価が高く感じる"のか？」
- 「🍈が高止まりしている時、他の通貨果物にどんな力がかかるのか？」

といった疑問を、PPPとミカン定理を通じて一貫して理解することができるのです。
