# 第8章 月次分析とPPP更新頻度の影響

## 8.1 月次観測の意義

第4章では年次データに基づく長期的な構造変化を分析した。以下に再掲する20年間のグラフ（図8.0）は、2011年の円高ピーク、2018年のトルコリラ危機、2022年のドル高局面という大きな構造変化を捉えている。

![年次MCI座標の推移（2005-2024）](annual_mci_plot.png)

*図8.0: 年次MCI座標の推移（第4章より再掲）。20年間の長期トレンドは把握できるが、直近数年の詳細な動態は読み取りにくい。*

しかし実際の市場参加者にとっては、より短い時間軸での変動把握が重要である。上図では2022年以降の動きが圧縮されており、月次の動態分析には不十分である。本章では2022年から2025年にかけての月次データを用い、MCI座標の短期的動態を分析する。

月次観測により、年次データでは平滑化されてしまう短期的なストレスの蓄積・解消過程、市場イベントに対する構造的反応速度、レジーム転換の兆候となる先行的な動きが明らかになる。

## 8.2 年単位PPPによる月次MCI

以下の図は、2022年1月から2025年11月までの月次MCI座標を示す。PPPは各年の確定値（IMF WEO）を使用している。

![月次MCI（年単位PPP）](monthly_mci_annual_ppp.png)

*図8.1: 年単位PPPを用いた月次MCI座標の推移。オレンジの点線は年境界（PPP更新タイミング）を示す。*

### 観察される特徴

このグラフから、いくつかの重要な特徴が読み取れる。

**1. 年境界での不連続性（軸のずれ）**

オレンジの点線で示された年境界において、MCI座標が急激にジャンプしている。これは為替レートが急変動したわけではなく、MCI算出の基準となるPPP値が年次更新されたことによる見かけ上の変化である。

特に顕著なのは2023年1月のm[TRY]が約-0.75から-0.45へ急上昇（+0.30）、2024年1月のm[TRY]が約-0.71から-0.43へ急上昇（+0.28）である。これらのジャンプは、トルコの高インフレを反映してIMFがPPP基準値を大幅に改定した結果、MCI座標を算出する際の「ものさし」自体が変わったことを示している。

**2. 年内の動態**

年境界間では、MCI座標は比較的滑らかに推移している。2022年後半のm[TRY]の低下（-0.6 → -0.75）は、PPP更新を待たずに進行した構造的ストレスの蓄積を示している。

**3. 情報の遅効性**

PPPは本質的に遅効指標である。IMFやWorld Bankの公式PPP推計は当該年の数値が翌年以降に公表され、遡及的に修正されることがあり、3年に一度の大規模調査（ICP）に基づき較正される。このため年単位PPPを用いた場合、実際の購買力変化がMCI座標に反映されるまでにタイムラグが生じる。

## 8.3 PPP補間による連続的推移

年境界での不連続性を解消するため、PPPを月次に線形補間したバージョンを作成した。

![月次MCI（補間PPP）](monthly_mci_interpolated_ppp.png)

*図8.2: 月次補間PPPを用いたMCI座標の推移。年境界でのジャンプが解消され、滑らかな推移を示す。*

### 補間方法

IMF WEOの年次PPP値は、その年の12月時点の購買力平価を表す。したがって月次補間は前年12月から当年12月への線形補間として実装する。

```
PPP(year, month) = PPP(year-1) + (PPP(year) - PPP(year-1)) × month / 12
```

ここで、`PPP(year-1)`は前年12月のPPP値（= 前年の年次PPP確定値）、`PPP(year)`は当年12月のPPP値（= 当年の年次PPP確定値または推定値）、`month`は当該月（1-12）である。

補間例（2024年）:
```
2024-01: PPP_JPY = 92.84 + (93.2 - 92.84) × 1/12 = 92.87
2024-06: PPP_JPY = 92.84 + (93.2 - 92.84) × 6/12 = 93.02
2024-12: PPP_JPY = 92.84 + (93.2 - 92.84) × 12/12 = 93.20
```

この補間により、年境界（12月→1月）でのジャンプは完全に解消される：
```
2023-12: JPY=92.84, TRY=8.074   (2023年12月値)
2024-01: JPY=92.87, TRY=8.447   (2024年12月へ向かって補間開始)
```

### 月次MCIグラフの特徴

**1. トレンドの明確化**

m[TRY]は-0.60付近（2022年初）から-0.47付近（2025年11月）へ、約3年かけて段階的に改善している。m[JPY]は+0.19付近（2022年初）から-0.01付近（2025年11月）へ継続的に低下し、m[USD]は+0.41〜+0.60の範囲で推移、2025年11月時点で+0.49である。

**2. 短期的なストレスの可視化**

2023年6月のシムシェク就任前後におけるm[TRY]の底打ち（-0.77付近）など、短期的なストレスイベントが明確に観察できる。

**3. ストレスと通貨下落の関係：トルコリラにおける構造転換**

ドル/リラ相場においてリラは一貫して下落を続けているが、MCIグラフではm[TRY]が0に向かって収束（改善）している。この一見矛盾した動きは、MCIが示すものの本質を理解することで説明できる。

MCIが映し出すのは「価格の動向」ではなく「構造的ストレスの在り方」である。円を例にとると、現在m[JPY]はほぼ0付近にあるが、これはPPPとの乖離率が0であることを意味しない。実際、円はPPP比較でいえば依然として割安な評価にある。MCIにおける0とは「バスケット内で構造的にバランスが取れた位置」を示す。

**リラの構造的ストレス蓄積期（2022年前半〜2023年7月）**

月次データによれば、m[TRY]は2022年1月の-0.595から継続的に悪化し、2023年7月に-0.766で底を打った。

- 2022年1月〜12月: -0.595 → -0.755（約-0.16の悪化）。エルドアン政権下での非正統的な金融政策が継続。高インフレ下での低金利固執と為替介入により市場メカニズムに逆行し、通貨を人工的に支える試みが構造的ストレスを蓄積した。

- 2023年1月〜5月: -0.738 → -0.637（一時的な改善傾向）。しかし構造的な政策転換には至らず、6月以降再び悪化。

- 2023年6月〜7月: -0.715 → -0.766（急激な悪化、底打ち）。2023年6月3日にシムシェク財務大臣が就任。就任直後、市場は政策転換の不確実性に直面し、m[TRY]は6月に-0.715へ、7月には-0.766で最低値を記録した。これは政策転換への過渡期における一時的なストレス極大化と解釈できる。

**構造転換と収束トレンドの形成（2023年8月〜）**

2023年8月以降、明確な収束トレンドが形成された。2023年8月〜12月は-0.744 → -0.711（改善開始）、2024年通年は-0.696 → -0.517（大幅改善）、2025年1月〜11月は-0.503 → -0.472（継続的改善）である。

シムシェク財務大臣のもとでインフレ率に見合った政策金利への正常化が進められた結果、リラの下落は「市場に抗う人工的な維持」から「インフレに見合った自然な調整」へと性質を変えた。価格は下がり続けているが、その下落が経済実態を反映したものとなったことで構造的ストレスは解消され、MCIは明確な収束傾向を示している。

底打ち（2023年7月）からの約2年半で、m[TRY]は-0.766から-0.472へと約0.29ポイント改善しており、年率約0.12ポイントの安定的な収束速度を示している。これは政策の一貫性と市場の信認回復を反映している。

## 8.4 先行指標としての月次MCI

月次データの分析から、MCIが持つ先行指標としての性質が示唆される。

### 構造改善の先行性

月次MCIグラフにおいてm[TRY]は2023年後半から緩やかな改善傾向を示している。これはPPPへの収束圧力（極端な割安状態は持続不可能であり、何らかの形で調整される）、構造的ストレスの解消（インフレ率の低下や金融政策の正常化が進行）、市場価格への反映遅延（構造改善が価格に反映されるまでにはタイムラグがある）を反映している。

MCIの改善傾向は、将来的な通貨安定化の兆候と解釈できる可能性がある。ただしこれは「収束時期」を予測するものではなく、「収束方向への構造的圧力」を示すものである。

### 限界

PPP自体が遅効指標であるため完全な先行性は期待できない。政策変更や地政学的イベントによる急変には対応できず、月次データでも日次・週次の短期変動は平滑化されている。

## 8.5 月次MCIによる将来価格帯の推定

### 可動域の限定性と統計的推定の可能性

Mikan 3-Currency clr Index の重要な数理的特性として、可動域の限定性（Boundedness）が挙げられる。実勢為替レートは理論上 $0 \to \infty$ の範囲で変動しうるため、将来の変動域を直接的に推測することは困難である。しかし $m[i]$ 座標はゼロサム制約（式(1)）により、数学的に制約された空間内に写像される。

月次補間データ（2022年1月〜2025年11月、47ヶ月）における $m[i]$ の観測範囲は以下の通りである。

| 通貨 | 最小値 | 最大値 | レンジ幅 |
|-----|--------|--------|----------|
| USD | +0.406 | +0.597 | 0.191 |
| JPY | -0.015 | +0.197 | 0.212 |
| TRY | -0.766 | -0.472 | 0.294 |

この観測範囲は、年次データ（20年間）で観測された $\pm 0.8$ 程度の範囲と比較して、より限定的である。月次補間データにおける変動は以下のように穏やかである。

| 通貨 | 最大月次上昇 | 発生時期 | 最大月次下落 | 発生時期 | 平均月次変化 |
|-----|------------|---------|-------------|---------|-------------|
| USD | +0.058 | 2023年6月 | -0.029 | 2024年8月 | 0.011 |
| JPY | +0.047 | 2024年8月 | -0.039 | 2022年4月 | 0.017 |
| TRY | +0.036 | 2023年2月 | -0.077 | 2023年6月 | 0.019 |

注目すべき発生時期として、TRY最大下落（2023年6月、-0.077）はシムシェク財務大臣就任月であり、政策転換期における構造的ストレスの極大化を反映している。JPY最大下落（2022年4月、-0.039）はFOMCのQT加速合意とCPI 8.5%発表により米長期金利が急騰（2.4%→3.0%）し、日米金利差の急拡大による円安圧力が生じた時期である。JPY最大上昇（2024年8月、+0.047）は日銀の金融政策転換期に対応し、USD最大上昇（2023年6月、+0.058）はTRYの大幅下落に伴うバスケット内での相対的上昇（ゼロサム制約の結果）である。

この限定性は、統計的手法を用いて $m[i]$ の短期的な理論的可動域を算出できる可能性を示唆する。

### 逆算による目標価格の設定

$m[i]$ の将来値に関する仮定を置くことで、将来の為替レートを以下の逆算式により推定できる。

$$
S_{\text{A/B,future}} = \text{PPP}_{\text{A/B,future}} \times \exp(m[A]_{\text{target}} - m[B]_{\text{target}})
$$

ここで $\text{PPP}_{\text{future}}$ は各国のインフレ率予測から推定可能であり、$m[i]_{\text{target}}$ は現在のトレンド外挿または統計的範囲から設定する。この手法により、単なる事後的評価にとどまらず、統計的根拠に基づく将来の価格帯推定が可能となる。

### 実践例：2025年12月のUSD/JPY価格帯推定

**現状の把握（2025年11月時点）**

月次補間PPPを用いた最新のMCI座標（2025年11月）は、$m[\text{USD}] = +0.487$（割高維持）、$m[\text{JPY}] = -0.015$（ほぼ中立）、$m[\text{TRY}] = -0.472$（割安）である。

**トレンド分析**

2023年以降の月次データから観察される傾向は以下である。USDは2023年の+0.597をピークに緩やかに低下し、2025年11月時点で+0.487。JPYは2022年の+0.197から継続的に低下し、2025年11月時点でほぼ中立（-0.015）。TRYは2023年7月の-0.766（底）から改善傾向で、2025年11月時点で-0.472である。

**シナリオ設定**

2025年11月実測値（USD: +0.487, JPY: -0.015, TRY: -0.472）を基準とし、観測されている統計的トレンドを2025年12月まで外挿する。

| シナリオ | m[USD] | m[JPY] | m[TRY] | 根拠 |
|---------|--------|--------|--------|------|
| A（慣性維持） | +0.485 | -0.025 | -0.460 | JPY低下・TRY収束の各トレンド継続 |
| B（やや円高） | +0.478 | 0.000 | -0.478 | JPYが0付近へ、USDは統計的比率（約60%）で連動下落 |
| C（やや円安） | +0.480 | -0.020 | -0.460 | JPY低下継続、TRY収束継続 |

**2025年12月の推定レンジ**

PPP推定値（JPY: 93.52, TRY: 16.51）と各シナリオのm値を用いて算出した結果は以下である。

| 通貨ペア | 中心シナリオ | リスクレンジ |
|---------|-------------|-------------|
| USD/JPY | 151〜156円 | 147〜162円 |
| USD/TRY | 42.3〜43.0リラ | 40.4〜44.7リラ |
| TRY/JPY | 3.51〜3.67円 | 3.28〜4.01円 |

中心シナリオは3つのシナリオから得られる範囲であり、リスクレンジは過去の月次変動（±0.05）を考慮した統計的変動範囲である。

この±0.05の変動幅は、観測期間中の最大級イベント—2022年4月のFOMCによるQT加速（JPY月次変動-0.039）や2024年8月の日銀政策転換（JPY月次変動+0.047）—に匹敵する規模である。つまりレジームチェンジ級の構造変化が発生した場合に到達しうる水準を示している。

2025年12月は米FRBによる追加利下げの可能性が市場で議論されている時期であり、その決定内容や規模が市場予想と大きく乖離した場合、リスクレンジ相当の変動が実現する可能性がある。

## 8.6 本章のまとめ

月次分析により、以下の知見が得られた。

短期動態の可視化により、月次データは年次データでは平滑化される短期的なストレス蓄積・解消過程を観察できる。政策転換の即時追跡として、シムシェク就任（2023年6月）前後の構造転換を月単位で捉えることができた。先行指標の可能性として、月次MCIの改善傾向は将来の価格調整を示唆する可能性がある。将来価格帯の推定として、統計的手法により短期的な為替レートの理論的変動範囲を導出できる。

本章で示した月次分析は、第7章で提示した実務的運用指針をより短い時間軸で補完するものである。年次データによる長期的な構造把握と、月次データによる短期的な動態追跡を組み合わせることで、Mikan 3-Currency clr Index の実用性はさらに高まるだろう。
