**日付**: 2025-11-20

## 問題の概要

論文の式(1)とそれに基づく全ての解釈が、実装されているデータセットと**符号が完全に反転**していることが判明しました。

実データ検証により、データセットの実装が正しく、論文の定義式と解釈を修正する必要があります。

## 修正箇所一覧

### 1. 第2章 理論枠組みと数理定義

#### 式(1)の修正

**誤（現在の論文）:**
```
m[USD] = -(d_USDJPY + d_USDTRY) / 3
m[JPY] = (2·d_USDJPY - d_USDTRY) / 3
m[TRY] = (2·d_USDTRY - d_USDJPY) / 3
```

**正（修正版）:**
```
m[USD] = (d_USDJPY + d_USDTRY) / 3
m[JPY] = (-2·d_USDJPY + d_USDTRY) / 3
m[TRY] = (d_USDJPY - 2·d_USDTRY) / 3
```

#### m[i]の解釈の修正

**誤（現在の論文）:**
> 正の値はその通貨がバスケット内で相対的に割安（低く評価されている）ことを、負の値は割高（高く評価されている）ことを示す

**正（修正版）:**
> **負の値**はその通貨がバスケット内で相対的に**割安**（低く評価されている）ことを、**正の値**は**割高**（高く評価されている）ことを示す

#### 差分関係の修正

**誤（現在の論文）:**
```
m[JPY] - m[USD] = d_USD→JPY
m[TRY] - m[USD] = d_USD→TRY
m[TRY] - m[JPY] = d_USD→TRY - d_USD→JPY
```

**正（修正版）:**
```
m[USD] - m[JPY] = d_USD→JPY
m[USD] - m[TRY] = d_USD→TRY
m[TRY] - m[JPY] = d_USD→TRY - d_USD→JPY
```

または等価的に：
```
m[JPY] - m[USD] = -d_USD→JPY
m[TRY] - m[USD] = -d_USD→TRY
```

### 4. 第4章・第7章 割安/割高の解釈を全て反転

以下の表現を全て逆転させる必要があります：

| 現在の誤った表現 | 正しい表現 |
|------------------|------------|
| "TRYの点が輪の**割安側**に大きく移動" | "TRYの点が輪の**割高側**に大きく移動" |
| "ドルと円は相対的に**割高方向**へ" | "ドルと円は相対的に**割安方向**へ" |
| "$m[\text{USD}]$が目立って**負の側**へ振れた（ドル**割高**）" | "$m[\text{USD}]$が目立って**正の側**へ振れた（ドル**割安**）" |
| "$m[\text{JPY}]$が大幅な**負の値**となり（円極度の**割高**）" | "$m[\text{JPY}]$が大幅な**正の値**となり（円極度の**割安**）" |
| "$m[\text{TRY}]$は大幅な**正の値**（リラ極度の**割安**）" | "$m[\text{TRY}]$は大幅な**負の値**（リラ極度の**割高**）" |

**重要**: 符号の解釈が逆転しても、経済的な実態の解釈（例：リラ暴落、円高、ドル高など）は変わりません。m[i]の符号の定義だけが変わります。

>
> $m_i(t) < 0$ のとき、通貨 $C_i$ は相対的に割高（高く評価されている）ことを示す。

**正（修正版）:**
> $m_i(t) > 0$ のとき、通貨 $C_i$ は三通貨バスケット内で相対的に**割高**（高く評価されている）ことを示す。
>
> $m_i(t) < 0$ のとき、通貨 $C_i$ は相対的に**割安**（低く評価されている）ことを示す。
