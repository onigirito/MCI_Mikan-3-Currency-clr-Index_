# 第8章 限界と今後の展望

## 8.1 本研究の限界

本研究では、Mikan 3-Currency clr Index という新しい通貨評価フレームワークを提案し、USD・JPY・TRY の3通貨について実証分析を行った。しかし、以下のような限界が存在する。

### 8.1.1 データの制約

#### 1. 時間解像度の制約

本研究では、**年次データ**（2005–2024年、20年間）のみを使用した。これには以下の限界がある：

- **短期的な変動の捕捉不可**：日次・週次・月次レベルでの急激な通貨変動（例：2022年10月のUSD/JPY 150円突破、2018年8月のリラ急落）を詳細に追跡できない

- **レジーム転換の時期特定精度**：年次データでは、転換が年内のいつ発生したかを特定できない。例えば、2013年のアベノミクス効果は4月以降に顕著だが、年次平均では緩和されて見える

- **高頻度トレーディングへの適用困難**：日次・時間次のデータを必要とする実務的な投資判断には直接適用できない

#### 2. 通貨数の制約

本研究は**3通貨（USD、JPY、TRY）**に限定されている。これには以下の問題がある：

- **グローバルな通貨構造の欠如**：EUR、GBP、CNY、AUD などの主要通貨が含まれていないため、世界全体の通貨評価構造を代表していない

- **地域的偏り**：USD（北米）、JPY（東アジア）、TRY（中東）という限定的な地域選択であり、欧州・南米・オセアニア等が欠落

- **3通貨の制約**：組成データ解析（CoDA）の枠組みでは、3通貨の場合は2次元平面への射影が自然だが、4通貨以上では次元が増え、可視化が複雑になる

#### 3. 基準レート（K_i）の選択

本研究では **PPP（購買力平価、GDPベース）** を基準レートとして採用したが、これには以下の限界がある：

- **PPPの理論的限界**：PPPは長期的な均衡概念であり、短期的な通貨評価には必ずしも適切でない。また、貿易財と非貿易財の比率、バスケット構成の違いなどにより、PPPそのものに測定誤差が含まれる

- **代替指標の可能性**：実質実効為替レート（REER）、ビッグマック指数、単位労働コスト（ULC）ベースのフェアレートなど、他の基準レートを使用すると、結果が異なる可能性がある

- **データ更新頻度**：World Bank の PPP データは年次更新であり、リアルタイム分析には向かない

### 8.1.2 モデルの前提と仮定

#### 1. 対数線形性の仮定

本手法では、d_ij = ln(S_ij / F_ij) という対数差分を基本量としている。これは以下を前提とする：

- **小幅な乖離への適合性**：対数近似は乖離率が小さい（例：±50%以内）場合に良好だが、極端な乖離（例：数倍の変動）では非線形効果が無視される

- **正規分布の仮定**：統計分析（平均、標準偏差、相関係数）は、対数差分がおおむね正規分布に従うことを暗黙に仮定している

#### 2. ゼロサム制約の解釈

本手法の核心である **m[USD] + m[JPY] + m[TRY] = 0** というゼロサム制約は、「3通貨の閉じたバスケット内での相対評価」を意味する。これには以下の限界がある：

- **外部通貨の影響**：EUR や CNY などバスケット外の通貨の影響は、直接観測できない（ShadowModel として間接的に反映されるのみ）

- **絶対的な評価値の欠如**：m[i] はあくまで「バスケット内での相対位置」であり、「USD が世界全体で割高か」という絶対的な評価ではない

#### 3. 静的な枠組み

本研究は各時点での **クロスセクション分析**（静的な相対評価）が中心であり、**動学的なモデル**（例：自己回帰モデル、誤差修正モデル）は構築していない。

- **因果関係の未検証**：m[i] の変化が何によって引き起こされるかという因果メカニズムは明示的にモデル化されていない

- **予測モデルの欠如**：将来の m[i] を予測する統計モデルは本研究の範囲外である

### 8.1.3 実証分析の範囲

#### 1. 記述統計とケーススタディ中心

本研究の実証分析は、主に記述統計（平均、標準偏差、相関）と特定年度のケーススタディ（2011年、2018年、2022年など）に基づいている。

- **統計的検定の不足**：レジーム転換の統計的有意性、構造変化の検定（Chow test 等）は実施していない

- **他の経済変数との関係**：金利差、経済成長率、インフレ率、財政赤字などのマクロ経済変数との関係は分析していない

#### 2. 単一の通貨トリプレット

本研究は USD-JPY-TRY という特定の組み合わせのみを分析しており、他の通貨トリプレット（例：USD-EUR-JPY、USD-CNY-INR）での妥当性は検証されていない。

### 8.1.4 実務適用への課題

#### 1. 取引コストの無視

本手法は、裁定取引のコストを考慮していない。実際の投資判断では、スプレッド、手数料、流動性リスクなどを考慮する必要がある。

#### 2. リスク調整の欠如

m[i] は「割高/割安」を示すが、そのポジションを取ることのリスク（ボラティリティ、最大ドローダウン等）は考慮されていない。

#### 3. 運用シグナルの精緻化不足

「m[i] > 0 なら割高だから売り、m[i] < 0 なら割安だから買い」という単純なルールの有効性は、バックテストで検証されていない。

## 8.2 今後の展望

上記の限界を踏まえ、以下の方向での拡張と深化が期待される。

### 8.2.1 データの拡張

#### 1. 高頻度データへの適用

**日次・月次データ**を用いた高頻度版 Mikan 3-Currency clr Index の構築が最優先課題である。

**実装方針**：
- **日次為替レート**：各国中銀、BIS、Investing.com などから取得
- **PPP の年次固定**：各年の PPP を年内で固定し、日次の S_ij(t) と組み合わせて d_ij(t) を算出
- **時系列プロット**：日次 m[i](t) をプロットし、短期的な変動パターンを分析

**期待される成果**：
- レジーム転換の正確な時期（月・日レベル）の特定
- 急激な市場イベント（例：Brexit、COVID-19 市場ショック）への反応の詳細観測
- 高頻度トレーディング戦略への適用可能性

#### 2. 通貨数の拡張

**4通貨以上**への拡張により、より包括的な通貨評価が可能になる。

**候補通貨の例**：
- **USD-JPY-EUR-GBP**（G4通貨）
- **USD-JPY-CNY-KRW**（東アジア通貨圏）
- **USD-EUR-GBP-CHF-JPY-AUD-CAD**（G7主要通貨）

**技術的課題**：
- **次元の増加**：4通貨では3次元、5通貨では4次元となり、可視化が困難になる。主成分分析（PCA）や t-SNE などの次元削減技術が必要
- **データ整合性**：多通貨の PPP データを同一ソースから一貫して取得することが重要

**期待される成果**：
- グローバルな通貨構造の全体像の把握
- 特定通貨ペアの孤立性・連動性の定量化

#### 3. 他の基準レート（K_i）の検討

PPP 以外の基準レートを用いた Mikan Index のバリアントを構築する。

**候補**：
- **実質実効為替レート（REER）ベース**：BIS や IMF のREERデータを使用
- **ビッグマック指数ベース**：消費財の価格比較に基づく
- **単位労働コスト（ULC）ベース**：労働生産性と賃金から算出される競争力指標
- **金利平価ベース**：金利差から理論的な為替レートを算出

**期待される成果**：
- 異なる基準レートによる結果の比較（ロバストネス検証）
- 特定の用途（例：貿易収支分析には ULC、消費財価格比較にはビッグマック）に最適な基準レートの選択

### 8.2.2 モデルの深化

#### 1. 動学モデルの構築

**時系列モデル**を導入し、m[i](t) の動的挙動を分析する。

**候補モデル**：
- **VAR（ベクトル自己回帰）モデル**：m[USD](t), m[JPY](t), m[TRY](t) の相互依存関係をモデル化
- **誤差修正モデル（ECM）**：長期均衡（m[i] の平均）からの乖離とその修正過程を分析
- **状態空間モデル**：レジーム転換を潜在状態として推定（Markov-switching model 等）

**期待される成果**：
- m[i](t) の予測モデルの構築
- レジーム転換の早期検出アルゴリズムの開発
- 因果関係の推定（Granger causality test 等）

#### 2. 機械学習との統合

**機械学習手法**を用いて、m[i](t) と他のマクロ経済変数の非線形関係を学習する。

**アプローチ例**：
- **回帰分析**：m[i](t+1) を予測するために、金利差、GDP成長率、インフレ率、VIXなどを説明変数とする
- **クラスタリング**：時系列パターンに基づいてレジームを自動分類（k-means、DBSCAN 等）
- **深層学習**：LSTM や Transformer を用いた時系列予測

**期待される成果**：
- より高精度な予測モデル
- 非線形効果や複雑な相互作用の発見

#### 3. 外部指標との統合フレームワーク

第6章で論じた ShadowModel の概念を、**外部指標（DXY、VIX、金利差等）**と統合するフレームワークを構築する。

**統合方法**：
- m[USD](t) と DXY(t) の回帰分析により、「外部ドル強度」の影響を定量化
- 残差分析により、「3通貨バスケット内部の独自変動」を抽出

**期待される成果**：
- 内部要因と外部要因の分離
- より包括的な通貨評価システムの構築

### 8.2.3 実証分析の深化

#### 1. 他の通貨トリプレットでの検証

USD-JPY-TRY 以外の通貨組み合わせで、本手法の妥当性を検証する。

**例**：
- **USD-EUR-JPY**（主要先進国通貨）
- **USD-CNY-JPY**（東アジア経済圏）
- **USD-BRL-MXN**（米州新興国通貨）

**期待される成果**：
- 本手法の一般性の検証
- 通貨圏ごとの特性の比較

#### 2. 経済イベントとの対応分析

特定の経済イベント（中銀の政策変更、金融危機、地政学的ショック）前後での m[i] の挙動を系統的に分析する。

**イベント例**：
- FRB の利上げサイクル（2015–2018、2022–）
- ECB の量的緩和開始・終了
- 英国 Brexit 投票（2016年6月）
- COVID-19 市場ショック（2020年3月）

**手法**：
- **イベントスタディ**：イベント前後のウィンドウでの m[i] の変化を統計的に検定
- **構造変化の検出**：Chow test、CUSUM test などでレジーム転換点を統計的に特定

**期待される成果**：
- m[i] の経済的解釈の深化
- 市場の「サプライズ」の定量化

#### 3. バックテスト分析

本手法に基づく**投資戦略**のバックテストを実施する。

**戦略例**：
- **平均回帰戦略**：m[i] が極端な値（例：±2標準偏差）に達したら反対ポジションを取る
- **モメンタム戦略**：m[i] の変化率（Δm[i]）が大きい方向にポジションを取る
- **ペアトレード**：m[i] 間の差（例：m[USD] - m[JPY]）が拡大したらペアトレードを仕掛ける

**評価指標**：
- シャープレシオ、最大ドローダウン、勝率、収益率

**期待される成果**：
- 実務的な有効性の検証
- リスク調整後のパフォーマンスの定量化

### 8.2.4 実務応用の拡大

#### 1. リスク管理への応用

多国籍企業や金融機関の**為替リスク管理**において、m[i] を以下のように活用する：

- **ヘッジ比率の決定**：m[i] が極端な値の場合、ヘッジ比率を引き上げる
- **エクスポージャーの再配分**：割安通貨へのエクスポージャーを増やし、割高通貨を減らす

#### 2. 中央銀行の政策分析

中央銀行が**自国通貨の評価水準**をモニタリングする際の補助指標として活用する。

**例**：
- 日銀が m[JPY] をモニタリングし、極端な円高（m[JPY] > +0.4）時には介入を検討
- トルコ中銀が m[TRY] の割安度（< -0.7）を改善するための政策を議論

#### 3. 学術研究への貢献

本手法は、**組成データ解析（CoDA）の経済学への応用例**として、以下の分野に貢献しうる：

- **国際金融論**：多通貨環境での為替レート決定理論
- **計量経済学**：ゼロサム制約下での時系列分析手法
- **データサイエンス**：組成データの可視化と解釈手法

## 8.3 結語

本研究では、Mikan 3-Currency clr Index という新しい通貨評価フレームワークを提案し、USD・JPY・TRY の3通貨について2005–2024年の実証分析を行った。

本手法の主要な貢献は、以下の3点にまとめられる：

1. **ゼロサム制約の導入**：組成データ解析の枠組みを用いて、三通貨の相対評価を整合的に表現
2. **多通貨同時評価**：二国間比較の限界を克服し、三通貨の構造を単一の座標系で可視化
3. **経済的解釈の妥当性**：2011年超円高、2018年リラ危機、2022年ドル高など、主要な経済イベントと整合的な結果

しかし、本研究は年次データ・3通貨・PPPという限定的な設定での初期的検証であり、多くの限界が存在する。今後、高頻度データへの拡張、通貨数の増加、動学モデルの構築、他の基準レートとの比較など、多方面での発展が期待される。

特に、**日次・月次データへの適用**と**4通貨以上への拡張**は、本手法の実務的有用性を大きく高めるものであり、最優先の課題である。

最終的には、Mikan 3-Currency clr Index が、為替市場の実務家、政策立案者、学術研究者にとって、**多通貨環境における通貨評価の標準的なツール**として確立されることを期待する。
