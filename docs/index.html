About MCI / MCIã«ã¤ã„ã¦

The Mikan 3-Currency clr Index (MCI) is a novel currency valuation framework that:

  • Uses Purchasing Power Parity (PPP) as the equilibrium baseline
  • Applies centered log-ratio (clr) transformation from Compositional Data Analysis (CoDA)
  • Evaluates any three currencies in a zero-sum basket structure (this paper uses USD, JPY, and TRY)
  • Identifies currency over/undervaluation relative to PPP fundamentals
