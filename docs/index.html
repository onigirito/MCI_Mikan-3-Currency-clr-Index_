Mikan 3-Currency clr Index (MCI)

A PPP-based structural valuation index for three currencies (USD, JPY, TRY) using centered log-ratio (clr) transformation from Compositional Data Analysis (CoDA).

PPP（購買力平価）を基準とした3通貨（USD・JPY・TRY）の構造的評価指標です。組成データ解析（CoDA）のclr変換を用いて、ゼロサム制約下での通貨の過大評価・過小評価を分析します。

About MCI / MCIについて

The Mikan 3-Currency clr Index (MCI) is a novel currency valuation framework that:

  • Uses Purchasing Power Parity (PPP) as the equilibrium baseline
  • Applies centered log-ratio (clr) transformation from Compositional Data Analysis (CoDA)
  • Evaluates USD, JPY, and TRY in a zero-sum basket structure
  • Identifies currency over/undervaluation relative to PPP fundamentals

MCIは、PPPを均衡基準とし、CoDAのclr変換を適用することで、3通貨間の相対的な過大評価・過小評価を構造的に捉える指標です。
